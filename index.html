This page will show your Google Calendar events side by side with athlete meet dates and program due dates — everything in one view.

What's needed to build this:
1. Move project to Claude Code
2. Set up a free Google Cloud project (~15 min)
3. Sign in with Google once to authorize
4. Your calendar appears here automatically

Database setup

Go to supabase.com, create a free project, go to Project Settings → API and paste your keys. Run this SQL in the SQL Editor:

create table athletes (id uuid primary key default gen_random_uuid(), name text, program text, due_date text, notes text, meets jsonb default '[]', start_date text, end_date text, churn_reason text, is_active boolean default true, created_at timestamptz default now());
